### คำอธิบายระบบสั้น ๆ
ระบบสำหรับบันทึกและจัดการทะเบียนหนังสือของห้องสมุดวิทยาลัย สามารถเพิ่ม ลบ แก้ไข และค้นหาหนังสือ โดยเก็บข้อมูลสำคัญ เช่น ISBN, ชื่อหนังสือ, ผู้แต่ง, หมวดหมู่, ปีที่พิมพ์ และสถานะการยืม

### ตาราง API Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/health` | ตรวจสอบสถานะการทำงานของระบบ (JSON: status, version) |
| GET | `/api/books` | ดึงข้อมูลรายการหนังสือทั้งหมด |
| GET | `/api/books/:id` | ดึงข้อมูลหนังสือรายเล่มตาม ID |
| POST | `/api/books` | เพิ่มข้อมูลหนังสือเล่มใหม่เข้าไปในระบบ |
| PUT | `/api/books/:id` | แก้ไขข้อมูลหนังสือ (รวมถึงการปรับสถานะ) |
| DELETE | `/api/books/:id` | ลบข้อมูลหนังสือออกจากระบบ |
