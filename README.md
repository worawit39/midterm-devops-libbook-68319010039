## 🛣️ ตาราง API Endpoints (Backend พอร์ต 3001)

| Method | Endpoint | Description |
| :--- | :--- | :--- |
| **GET** | `/health` | ตรวจสอบสถานะการทำงานของ API (ส่งกลับข้อมูล JSON `{ status, version }`) |
| **GET** | `/api/books` | ดึงข้อมูลรายการหนังสือทั้งหมดในห้องสมุด |
| **GET** | `/api/books/:id` | ดึงข้อมูลรายละเอียดของหนังสือแบบระบุ ID รายเล่ม |
| **POST** | `/api/books` | เพิ่มข้อมูลหนังสือใหม่เข้าสู่ระบบ |
| **PUT** | `/api/books/:id` | แก้ไข/อัปเดตข้อมูลหนังสือที่มีอยู่ในระบบตาม ID |
| **DELETE** | `/api/books/:id` | ลบข้อมูลหนังสือออกจากระบบตาม ID |
